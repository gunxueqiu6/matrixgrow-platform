```
Webhook (POST /ai-rewrite)
 → Input Parser
 → Load Agent Prompts
 → Platform Router（5 路 Switch）
 → 4 × AI Agent（Claude/GPT HTTP Request）
 → Merge Results（multiplex）
 → Format Output
 → Response
```

约 15 个节点。可被 core-webhook.json 通过 Execute Workflow 节点调用。

**2.2 创建 `n8n-workflows/flows/multi-publish.json`**（全平台分发子工作流）：

```
Webhook (POST /multi-publish)
 → Platform Dispatch Router（Switch 按渠道类型分）
   ├─ API 分支（5 节点）: X, LinkedIn, Dev.to, Medium, Mastodon
   ├─ RPA 分支（17 节点）: 全部 RPA 平台
   └─ Artipub 分支（7 节点）: Dev.to/Medium fallback + 掘金/CSDN/思否/开源中国/头条号
 → Merge → Response
```

约 35 个节点。每个发布平台一个 HTTP Request 节点，调用 MatrixGrow API。
